Fiches de révision — Cybersécurité & Code du Numérique Bénin

Informations système
imageinfo / kdbgscanIdentification
vol.py -f dump.mem imageinfo

Identifie le profil du système (OS, version, architecture). Toujours la première commande à lancer sur un dump.

hivelistRegistre
vol.py -f dump.mem --profile=Win7SP1x64 hivelist

Liste les ruches (hives) du registre Windows chargées en mémoire (SAM, SYSTEM, SOFTWARE…).

printkeyRegistre
vol.py -f dump.mem --profile=Win7SP1x64 printkey -K "SOFTWARE\Microsoft\Windows\CurrentVersion\Run"

Affiche le contenu d'une clé de registre spécifique. Utile pour détecter la persistance de malwares.

Processus
pslistProcessus
vol.py -f dump.mem --profile=Win7SP1x64 pslist

Liste les processus actifs. Affiche PID, PPID, nom, heure de démarrage. Peut être trompé par les rootkits (ne montre que la liste chaînée officielle).

psscanProcessus
vol.py -f dump.mem --profile=Win7SP1x64 psscan

Scanne la mémoire physique pour trouver les structures EPROCESS. Détecte les processus cachés que pslist ne voit pas.

pstreeProcessus
vol.py -f dump.mem --profile=Win7SP1x64 pstree

Affiche les processus sous forme d'arbre parent-enfant. Pratique pour détecter des processus malveillants se faisant passer pour des processus légitimes.

cmdlineProcessus
vol.py -f dump.mem --profile=Win7SP1x64 cmdline

Affiche la ligne de commande utilisée pour lancer chaque processus. Révèle les arguments suspects.

procdumpExtraction
vol.py -f dump.mem --profile=Win7SP1x64 procdump -p 1234 --dump-dir /output/

Extrait l'exécutable d'un processus donné (par PID) depuis la mémoire vers le disque pour analyse.

Réseau
netscanRéseau
vol.py -f dump.mem --profile=Win7SP1x64 netscan

Affiche toutes les connexions réseau (TCP/UDP), actives ou fermées, avec IP locale/distante et PID associé.

connections / connscanRéseau
vol.py -f dump.mem --profile=WinXPSP2x86 connections

Pour Windows XP uniquement. Liste les connexions TCP actives. connscan scanne aussi les connexions terminées.

Mémoire & Fichiers
malfindDétection malware
vol.py -f dump.mem --profile=Win7SP1x64 malfind

Plugin phare pour détecter les injections de code (shellcode, DLL injection). Recherche les zones mémoire avec des permissions RWX et du contenu PE.

memdumpExtraction
vol.py -f dump.mem --profile=Win7SP1x64 memdump -p 1234 --dump-dir /output/

Extrait l'espace mémoire complet d'un processus. Utile pour récupérer des artefacts (clés, mots de passe, données chiffrées).

filescanFichiers
vol.py -f dump.mem --profile=Win7SP1x64 filescan

Scanne les objets FILE_OBJECT en mémoire. Permet de lister tous les fichiers ouverts au moment du dump, même supprimés.

dumpfilesFichiers
vol.py -f dump.mem --profile=Win7SP1x64 dumpfiles -Q 0xaddr --dump-dir /output/

Extrait un fichier depuis la mémoire à partir de son adresse (obtenue avec filescan).

Utilisateurs & Authentification
hashdumpCredentials
vol.py -f dump.mem --profile=Win7SP1x64 hashdump

Extrait les hashes NTLM des comptes utilisateurs depuis les ruches SAM et SYSTEM. Utilisé pour des attaques pass-the-hash.

lsadumpCredentials
vol.py -f dump.mem --profile=Win7SP1x64 lsadump

Extrait les secrets LSA (mots de passe de services, comptes de domaine, clés de chiffrement).

Loi n° 2017-20 — Code du NumériqueTexte fondateur

Adoptée le 20 avril 2018 au Bénin. Régit les transactions électroniques, la protection des données personnelles, la cybersécurité, la signature électronique et les services de communication électronique.

Protection des données personnelles
Art. 536–570Données personnelles

Principe du consentement éclairé

Toute collecte de données personnelles requiert le consentement libre, exprès et éclairé de la personne concernée. Les données doivent être collectées pour des finalités déterminées, explicites et légitimes.

Art. 552Données personnelles

Droit d'accès et de rectification

Toute personne a le droit d'accéder à ses données personnelles détenues par un organisme et d'en demander la rectification ou la suppression.

Art. 558–560Données personnelles

Transfert de données hors Bénin

Interdit sauf si le pays destinataire assure un niveau de protection adéquat ou si la personne concernée a donné son accord.

Cybercriminalité & Infractions
Art. 551–600Cybercriminalité

Accès frauduleux à un système (STAD)

Tout accès ou maintien frauduleux dans un système de traitement automatisé de données (STAD) est puni de 6 mois à 2 ans d'emprisonnement et/ou d'une amende de 500 000 à 2 000 000 FCFA.

Code du NumériqueCybercriminalité

Atteinte à l'intégrité des données

Toute introduction, modification, suppression ou altération frauduleuse de données est sanctionnée. Peines aggravées si l'infraction cible des systèmes de l'État ou des infrastructures critiques.

Code du NumériqueCybercriminalité

Interception illicite

L'interception non autorisée de communications électroniques est une infraction pénale. Elle est distincte de l'écoute légale autorisée par un juge.

Transactions & Commerce électronique
Art. 236–280Commerce élec.

Signature électronique

La signature électronique a la même valeur juridique que la signature manuscrite à condition d'être fiable, sécurisée et certifiée par un prestataire de services de certification accrédité.

Art. 281–320Commerce élec.

Contrat électronique

Un contrat conclu par voie électronique a pleine valeur légale. L'offre et l'acceptation peuvent être échangées par courrier électronique ou formulaire en ligne.

Institutions clés
APDPInstitution

Autorité de Protection des Données Personnelles — organisme indépendant chargé de veiller à l'application des règles de protection des données personnelles au Bénin. Créé par le Code du Numérique.

ARCEP BéninInstitution

Autorité de Régulation des Communications Électroniques et de la Poste — régule les opérateurs télécom et internet, délivre les licences, contrôle la qualité de service.

CERT BéninInstitution

Équipe nationale de réponse aux incidents de cybersécurité. Chargée de coordonner la réponse aux cyberattaques affectant les infrastructures béninoises.

Clique sur une question pour voir la réponse.

Quelle est la première commande à lancer lors de l'analyse d'un dump mémoire avec Volatility ?

Clique pour voir la réponse

imageinfo — elle identifie le profil système (OS, version, architecture) nécessaire pour toutes les commandes suivantes.
Quelle est la différence entre pslist et psscan ?

Clique pour voir la réponse

pslist parcourt la liste chaînée officielle des processus — un rootkit peut s'y cacher. psscan scanne directement la mémoire physique pour trouver les structures EPROCESS, détectant ainsi les processus masqués.
Quel plugin Volatility est utilisé pour détecter les injections de code malveillant ?

Clique pour voir la réponse

malfind — il recherche les zones mémoire avec des permissions RWX (lecture-écriture-exécution) contenant des signatures PE suspectes.
Quelle est l'année d'adoption du Code du Numérique du Bénin ?

Clique pour voir la réponse

Le Code du Numérique (Loi n° 2017-20) a été adopté et promulgué en 2018 (20 avril 2018).
Quelle institution béninoise est chargée de la protection des données personnelles ?

Clique pour voir la réponse

L'APDP — Autorité de Protection des Données Personnelles, créée par le Code du Numérique.
Quelle peine est prévue pour un accès frauduleux à un STAD au Bénin ?

Clique pour voir la réponse

6 mois à 2 ans d'emprisonnement et/ou une amende de 500 000 à 2 000 000 FCFA selon le Code du Numérique.
Que fait la commande hashdump dans Volatility ?

Clique pour voir la réponse

Elle extrait les hashes NTLM des comptes utilisateurs Windows depuis les ruches SAM et SYSTEM en mémoire. Ces hashes peuvent ensuite être crackés ou utilisés dans une attaque pass-the-hash.
Qu'est-ce que netscan affiche dans Volatility ?

Clique pour voir la réponse

Toutes les connexions réseau TCP/UDP (actives et terminées) avec les adresses IP locales/distantes, les ports et le PID du processus associé.
La signature électronique a-t-elle une valeur légale au Bénin ?

Clique pour voir la réponse

Oui — le Code du Numérique (art. 236–280) lui confère la même valeur qu'une signature manuscrite, à condition qu'elle soit fiable, sécurisée et certifiée par un prestataire accrédité.
